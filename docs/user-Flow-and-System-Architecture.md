# 사용자 흐름 & 시스템 구조


# 👣 사용자 여정 (User Flow)

```html
**[회원가입]
  ↓
[레벨테스트 (3~5분)]
  ↓
[퀵 스타트 카드 제공 (직무별 핵심 표현)]
  ↓
[추천 학습 홈 진입]
  ↓
[직무 시나리오 학습] + [레벨 기반 커리큘럼 병행]
  ↓
[복습 / 개인화 추천]**
```

---

# ⚡ 퀵 스타트 카드란?

- **사용자가 선택한 직무에 따라 제공되는 “오늘 꼭 필요한 5문장” 카드**
- **카드당 구성:**
    - **✅ 문장 (예: “안전모를 쓰세요.”)**
    - **🎧 TTS 음성 재생**
    - **🎤 사용자 발화(STT 인식)**
    - **🔁 피드백 및 반복 버튼**
    - **🇻🇳 베트남어 설명 포함**
- **레벨과 관계없이, 바로 써야 하는 실용 표현을 처음 접속하자마자 연습할 수 있게 함**

---

# 🔀 직무 시나리오 + 레벨 커리큘럼 병행 방식

- **레벨 기반 커리큘럼 (성장 중심)**
    - **레벨 0~4까지 구조화된 콘텐츠**
    - **초급 → 중급으로 문법/어휘/표현이 점차 확장됨**
    - **매일 1~2 레슨씩 추천 (학습 진도 저장)**
- **직무별 시나리오 (즉시 적용 중심)**
    - **선택한 직무(예: 공장, 건설)에 맞춘 “상황별 대화 시나리오”**
    - **예: “감독이 안전 지시를 한다” → 역할극 대화 → 발화 피드백**

- **즉, 사용자 입장에서는 레벨 콘텐츠 1개 + 직무 시나리오 1개를 매일 병행하도록 큐레이션 → 장기적 성장 + 현장 실용성을 함께 잡는 구조**

# 🧰 시스템 구성 개요

| **구성 요소** | **설명** |
| --- | --- |
| **🖥️ 웹 클라이언트** | **Vue.js 기반 SPA, 반응형 디자인, 다국어 지원** |
| **🔄 WebSocket 서버** | **음성 입력 → 실시간 스트림 처리 (낮은 지연)** |
| **🎙️ STT API** | **RTZR 또는 Naver CLOVA 사용 (베트남 화자 인식률 고려)** |
| **🧠 GPT 역할극 엔진** | **GPT-4 API 활용, 대화 흐름 + 피드백 생성** |
| **🔊 TTS API** | **OpenAI 또는 Naver TTS 사용, 속도 조절 가능** |
| **🧾 MySQL** | **사용자 정보, 학습 기록, 정답률 저장 및 조회** |
