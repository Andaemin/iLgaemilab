# 3분 레벨테스트 설계


# 🎯 목적

- **사용자의 한국어 이해도 및 발화 능력 초기 진단**
- **자동 채점 → 5단계 레벨(0~4) 매핑 → 추천 학습 경로 제공**

---

# 🧱 테스트 구성

| **유형** | **문항 수** | **배점** | **구현 방식** |
| --- | --- | --- | --- |
| **듣기 (MCQ)** | **4문제** | **1점 × 4 = 4점** | **오디오 + 객관식 보기 선택** |
| **읽기 이해** | **2문제** | **1점 × 2 = 2점** | **문장 + 보기 선택** |
| **단답형** | **2문제** | **2점 × 2 = 4점** | **문장 빈칸 또는 간단 문장 작성** |
| **말하기** | **2문제** | **5점 × 2 = 10점** | **음성 녹음(STT) + GPT 채점** |
- **총점: 20점 (자동 채점), 테스트 시간: 약 3~5분 소요**

---

# 🧠 문항 예시 (각 파트별 샘플)

### **✅ [1~4] 듣기 (Listening · 객관식 4문제, 각 1점)**

> **✅ 유형: 오디오 재생 → 보기 중 정답 선택**
> 
> 
> **✅ 채점: 사용자 선택 = 정답이면 1점**
> 

**문제 1**

🎧 오디오: "이쪽으로 오세요."

**보기**

① 오세요 ② 가세요 ③ 기다리세요 ④ 드세요

**정답**

: ① 오세요

📝 설명: "오세요"는 사람을 부를 때 쓰는 표현.

---

**문제 2**

🎧 오디오: "지금 점심시간이에요."

**보기**

① 아침이에요 ② 점심이에요 ③ 저녁이에요 ④ 회의해요

**정답**

: ② 점심이에요

📝 설명: "점심시간"이 들려야 함.

---

**문제 3**

🎧 오디오: "여기 위험해요. 조심하세요."

**보기**

① 위험해요 ② 늦었어요 ③ 시끄러워요 ④ 예뻐요

**정답**: ① 위험해요

📝 설명: '위험'이라는 단어를 듣고 선택해야 함.

---

**문제 4**

🎧 오디오: "화장실이 어디에 있어요?"

**보기**

① 언제 가요? ② 어디 있어요? ③ 누구 만나요? ④ 왜 가요?

**정답**

: ② 어디 있어요?

📝 설명: 질문의 유형은 위치 묻기 (어디)

---

### **✅ [5~6] 읽기 (Reading · 객관식 2문제, 각 1점)**

> **✅ 유형: 문장을 읽고 보기 중 정답 선택**
> 
> 
> **✅ 채점: 선택 = 정답이면 1점**
> 

**문제 5**

📝 문장: “사장님이 오늘 몇 시에 오십니까?”

**보기**

① 사람 ② 시간 ③ 장소 ④ 이유

**정답**

: ② 시간

📝 설명: “몇 시에” → 시간을 묻는 질문

---

**문제 6**

📝 문장: “이 기계를 끄고 자리를 정리하세요.”

**보기**

① 기계 사용 ② 물건 정리 ③ 불 끄기 ④ 안전 수칙

**정답**: ② 물건 정리

📝 설명: "자리를 정리하세요" → 정리 행동 강조

---

### **✅ [7~8] 단답형 (Short Answer · 2문제, 각 2점)**

> **✅ 유형: 빈칸 또는 직접 입력**
> 
> 
> **✅ 채점: 키워드 일치 + 문법 체크**
> 

**문제 7**

📝 문장: “이것을 ______ 주세요.”

**정답**

: 옮겨

📝 설명: 자연스러운 명령 표현 → "옮겨 주세요"

---

**문제 8**

📝 문장: “오늘 날씨가 좋아서 저는 _____ 갔어요.”

**정답**

: 공원에 / 밖에 / 산책하러 등

📝 설명: 의미상 연결 가능한 단어 허용 (부분 정답 허용 가능)

---

### **✅ [9~10] 말하기 (Speaking · 녹음형 2문제, 각 5점)**

> **✅ 유형: 사용자 음성 입력 → STT로 변환 → GPT 채점**
> 
> 
> **✅ 채점 항목: 발음(0–2) + 문법(0–2) + 단어/의사 전달(0–1)**
> 

**문제 9**

🎤 질문: "자기소개를 해보세요."

**예시 정답**

: “안녕하세요. 저는 안대민입니다. 베트남에서 왔습니다.”

📝 설명: 간단한 인사 + 이름 + 출신 정보 포함하면 정답

---

**문제 10**

🎤 질문: “기계를 어떻게 꺼요?”

**예시 정답**: “버튼을 눌러서 꺼요.”, “코드를 빼요.”

📝 설명: 동사 표현 + 문장 구조만 정확하면 정답 인정

---

# 🎯 점수 분포 및 레벨 매핑

| **점수 범위** | **매핑 레벨** | **설명** |
| --- | --- | --- |
| **0–4점** | **레벨0** | **자모/생존어 수준** |
| **5–9점** | **레벨1** | **초급: 인사, 요청 등** |
| **10–14점** | **레벨2** | **초중급: 요청·이해 가능** |
| **15–17점** | **레벨3** | **중급: 문장 생성 가능** |
| **18–20점** | **레벨4** | **고급: 유창한 응답 가능** |
- **이 기준은 TOPIK·세종학당 기준과의 간이 매핑을 고려함 → 정확성보단 실용적 시작 지점 추천이 목표**

---

# ✅ 구현 메모 (개발 참고사항)

- **음성 질문은 미리 녹음된 오디오로 제공 (MP3 or TTS)**
- **말하기 채점은 STT confidence + GPT 프롬프트 기반으로 구현**
- **레벨 결과는 테스트 직후 즉시 화면에 표시 → 이후 추천 학습 경로 카드와 연결됨**
