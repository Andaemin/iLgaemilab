# 레벨 기반 커리큘럼 구조

# 🧱 공통 레슨 단위 구성

```html
**[1] 목표 소개 (30초)
→ [2] 핵심 어휘 소개 (5~10개 단어)
→ [3] 예문 제공 (2~3개 짧은 문장)
→ [4] 음성 역할극 (시스템 ↔ 사용자 대화 3~5턴)
→ [5] 자동 피드백 제공 (발음/문법)
→ [6] 퀴즈 (객관식 or 발화 반복)**
```

# 🎯 레벨별 상세 구성 예시

### 🟩 레벨 0 — 생존어 & 문자 단계

| **구성** | **예시** |
| --- | --- |
| **🎯 목표** | **한글 자모 이해, 숫자 말하기, 간단한 인사 표현 학습** |
| **📘 핵심 어휘** | **가, 나, 다, 1, 2, 3, 안녕하세요, 감사합니다** |
| **🗣️ 예문** | **“안녕하세요.” / “감사합니다.” / “하나, 둘, 셋”** |

**🎭 역할극**

- 시스템: “안녕하세요.”
- 사용자: “안녕하세요.” (STT 인식)
- GPT 피드백: “'요' 발음이 조금 약해요.”

### 🟨 레벨 1 — 초급 단계

| **구성** | **예시** |
| --- | --- |
| **🎯 목표** | **기본 생활 표현 / 직장 내 지시 듣기** |
| **📘 핵심 어휘** | **주세요, 출근하다, 늦다, 주문하다, 여기요** |
| **🗣️ 예문** | **“이거 주세요.” / “출근했어요.”** |

**🎭 역할극**

- 시스템: “무엇을 도와드릴까요?”
- 사용자: “이거 주세요.”
- GPT 피드백: “조사가 정확해요. 발음도 좋아요.”

### 🟦 레벨 3 — 중급 단계

| **구성** | **예시** |

### 🟪 레벨 4 — 고급 단계
