담당자: 한승현 
진행 상태: 완료
마감일: 2025년 10월 24일
프로젝트: Planning (https://www.notion.so/Planning-2673e1faaf3581c59037d2951a4c309e?pvs=21)
우선순위: 높음
최종 편집자: 호 지

# ✅ 목적

- **직무별로 현장에서 바로 사용할 수 있는 표현과 대화를 학습하는 “상황 중심 학습 콘텐츠” 설계**
- **사용자의 레벨과 관계없이 실용적인 말하기 연습을 제공**
- **레벨 기반 커리큘럼과 병행하여 사용자의 언어 실력 + 직무 적응력을 동시에 향상시킴**

---

# 🎯 “레벨 기반 커리큘럼”과의 차이

| **구분** | **레벨 기반 커리큘럼** | **직무 시나리오** |
| --- | --- | --- |
| **목적** | **체계적 언어 능력 성장** | **현장 즉시 적용 가능한 표현 학습** |
| **대상** | **사용자 레벨별 콘텐츠 제공 (0~4)** | **직무 선택에 따라 공통 제공** |
| **내용** | **문법, 표현, 어휘 중심** | **상황 대화, 지시, 질문 응답 등** |
| **형식** | **단원별 학습 모듈** | **역할극 기반 시나리오 (TTS/STT/GPT)** |
| **추천 방식** | **레벨테스트 결과에 따라 자동 추천** | **회원가입 시 선택한 직무 기반 자동 제공** |
| **학습 방식** | **하루 1레슨 씩 순차 진도** | **하루 1시나리오 씩 큐레이션** |
|  |  |  |
- **레벨 커리큘럼은 언어 실력을 쌓는 학습 ↔ 직무 시나리오는 오늘 당장 쓸 수 있는 표현 연습**

---

# 🧩 병행 학습 방식

| **화면 구성 방식** | **설명** |
| --- | --- |
| **화면 내 병렬 구성** | **오늘의 학습 영역에서 레벨 기반 콘텐츠 + 직무 시나리오 카드를 함께 노출** |

***예시 UI***

```html
**📘 오늘의 학습

[🔹 레벨1 - 요청 표현]        [🔸 제조업 - 안전지시 시나리오]

TTS → 발화 → 피드백           역할극 → STT → GPT 평가**
```

***일일 학습 흐름***

```html
**1. [레벨 기반 학습] 진행 (레벨에 따라 콘텐츠 고정)
2. [직무 시나리오 학습] 진행 (직무에 따라 시나리오 변경)
3. 둘 다 완료 시 '복습' 모듈 진입 가능**
```

# **🎭 시나리오 구성 형식 (역할극 기반)**

| **구성 요소** | **예시** |
| --- | --- |
| **목표** | **"작업장 입장 전 안전 수칙 듣기 + 대답하기"** |
| **시스템 대사** | **“안전모를 쓰세요.” (TTS)** |
| **사용자 발화** | **“안전모를 쓰세요.” (STT 인식)** |
| **GPT 평가** | **“정확하게 말했어요. 발음 좋습니다.”** |
| **피드백 요소** | **발음 / 조사 / 속도 / 반복 버튼** |
- **모든 시나리오는 3~5턴 구성의 짧은 역할극이며, 사용자의 발화에 따라 즉시 피드백을 생성**

# 🛠 개발 설계 포인트

| **항목** | **설계 방식** |
| --- | --- |
| **추천 기준** | **회원가입 시 선택한 직무에 따라 자동 추천** |
| **콘텐츠 구조** | **`직무 → 상황 → 문장 리스트 → 역할극` 형태로 콘텐츠 관리** |
| **시나리오 분기** | **사용자의 발화 정확도에 따라 다음 질문 제시** |
| **발화 채점** | **GPT가 예상 발화와 비교해 점수 + 교정 제시** |
| **다국어 설명** | **피드백에 베트남어 번역 포함 (`translation_vn`)** |
