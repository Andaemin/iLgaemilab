# ✅ 목적

- **회원가입 & 레벨테스트 직후, 사용자에게 가장 먼저 보이는 학습 콘텐츠**
- **직무별 "오늘 꼭 써야 할 5문장"을 카드 형태로 제시**
- **사용자에게 빠른 실용성 + 성취감을 주고, 음성 기반 반복 연습 루틴을 유도**

---

# 🎯 사용자 흐름 내 위치

```html
**[회원가입]
  ↓
[레벨테스트 (3~5분)]
  ↓
▶ [퀵 스타트 카드 제공 (직무별 핵심 표현)]
  ↓
[추천 학습 홈 (레벨 + 직무 병행)]**
```

---

# 🧱 카드 구성 구조

| **구성 요소** | **설명** |
| --- | --- |
| **💬 문장** | **현장에서 실제 사용하는 짧은 문장 1개** |
| **🎧 TTS 버튼** | **원어민 음성 출력 (OpenAI or Naver TTS)** |
| **🎤 따라 말하기** | **사용자가 녹음 → STT 인식** |
| **✅ 피드백** | **GPT가 발음/문법 교정 (점수 + 설명)** |
| **🇻🇳 베트남어 번역** | **하단에 자동 제공** |
| **🔁 반복 학습** | **“다시 말하기” / “다음 문장” 버튼 포함** |
| **🎯 아이콘/그림** | **직관적인 상황 시각화 (예: 🪖, ⚙️, 🧯 등)** |
- **각 카드 = 1문장 단위로 구성되어 5장 제공, 카드 간 스와이프 또는 리스트 선택으로 이동 가능**

# 💬 카드 예시 (직무별)

### 🔧 제조직 예시

| **문장** | **베트남어 번역** | **아이콘** |
| --- | --- | --- |
| **“안전모를 쓰세요.”** | **Hãy đội mũ bảo hộ.** | **🪖** |
| **“이 기계를 꺼 주세요.”** | **Tắt máy này giúp tôi.** | **⚙️** |
| **“지금 쉬세요.”** | **Hãy nghỉ bây giờ.** | **🛌** |
| **“도와드릴게요.”** | **Tôi sẽ giúp bạn.** | **🤝** |
| **“여기로 오세요.”** | **Hãy đến đây.** | **👋** |

# 🛠 기능/개발 설계 포인트

| **항목** | **구현 방식** |
| --- | --- |
| **직무 태그 매핑** | **회원가입 시 선택한 직무 정보로 퀵스타트 카드 자동 필터링** |
| **데이터 구조** | **직무별 카드 세트 JSON or DB (5문장 단위)** |
| **GPT 피드백 처리** | **STT 결과 → 예상 문장 비교 → 발음/조사 피드백 제공** |
| **언어 지원** | **TTS: 느린 속도 설정 가능 / UI: 기본 베트남어, 한국어 보조 설명** |
| **재사용성** | **향후 레벨0~1 사용자에게 복습 카드로도 반복 제공 가능** |
