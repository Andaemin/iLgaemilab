# 실시간 대화형 학습 구조


# 🧭 기능 흐름

```html
**🎤 마이크 입력
   ↓
🗣️ STT (음성 → 텍스트 변환)
   ↓
🤖 GPT (상황 대화 시나리오 생성)
   ↓
🔊 TTS (응답 음성으로 출력)
   ↓
📋 피드백 표시 (교정 + 점수 + 반복)

사용자는 실제로 한국어로 말하면, 시스템이 즉시 대화 응답 + 교정 피드백을 제공**
```

---

# ⚙️ 기술 요소 요약

| **요소** | **설명** |
| --- | --- |
| **STT** | **RTZR 또는 Naver CLOVA API (실시간 음성 인식)** |
| **GPT** | **역할극 대화 생성 및 발화 피드백 담당** |
| **TTS** | **OpenAI Whisper / Naver Voice로 자연스러운 발화** |
| **WebSocket** | **STT ↔ GPT ↔ TTS 간 실시간 양방향 흐름 유지** |

---

# 🧠 GPT 시스템 프롬프트 예시

```json
{
  "turn": 2,
  "expected_text": "안전모를 쓰세요.",
  "correction": "‘를’ 조사가 빠졌어요.",
  "hint_vn": "Hãy đội mũ bảo hộ.",
  "score": 3,
  "next_prompt": "작업장 안으로 들어오세요."
}
```

---

# ✅ 자동 피드백 항목

- **발음 정확도 (STT confidence + GPT 평가)**
- **조사·어미 오류 교정 ("쓰요" → "쓰세요")**
- **어휘 누락 감지**
- **간단한 베트남어 설명 추가**
- **즉시 반복 버튼으로 다시 말하기 유도**
