# 이주 노동자 한국어 교육 MVP - 기능 명세서

## 🎯 프로젝트 목표
이주 노동자들의 한국어 학습을 돕는 실용적이고 효과적인 교육 플랫폼 구축

## 📋 현재 구현된 핵심 기능

### 1. 실시간 발음 교정 시스템 ✅
- **기술**: RTZR STT API + OpenAI GPT-4o-mini
- **기능**: 목표 문장과 사용자 발음 비교 분석
- **평가 요소**: 정확도 점수 (0-100점), 상세 피드백, 개선 제안
- **실시간성**: WebSocket 기반 즉시 음성 인식

### 2. 음성 인식 및 시각화 ✅
- **실시간 음성 레벨** 표시
- **부분 인식 결과** (실시간)
- **최종 인식 결과** (확정)
- **다국어 지원** (한국어/영어)

## 🚀 추가 구현 제안 기능

### 1. 직업별 맞춤 학습 콘텐츠
**우선순위: 높음**

#### 1.1 건설업 한국어
```javascript
// 예시 학습 콘텐츠
const constructionPhrases = [
  { text: "안전모를 써주세요", category: "안전", difficulty: 1 },
  { text: "시멘트를 가져다 주세요", category: "작업지시", difficulty: 2 },
  { text: "오늘 작업은 여기까지입니다", category: "일상대화", difficulty: 2 }

```

#### 1.2 제조업 한국어
- 기계 조작 용어
- 품질 관리 표현
- 시간 관리 대화

#### 1.3 서비스업 한국어  
- 고객 응대 표현
- 주문 받기 용어
- 친절 서비스 문구

### 2. 상황별 대화 시뮬레이션
**우선순위: 높음**

#### 2.1 일상생활 시나리오
- **병원 방문**: "어디가 아프세요?", "진료 예약하고 싶어요"
- **마트 쇼핑**: "이것 얼마예요?", "카드로 계산해 주세요"  
- **교통 이용**: "지하철역이 어디예요?", "버스 몇 번 타야 해요?"

#### 2.2 직장 내 상황
- **상사와 대화**: "네, 알겠습니다", "언제까지 하면 될까요?"
- **동료와 소통**: "도움이 필요해요", "같이 점심 먹을까요?"
- **고객 응대**: "죄송합니다", "다시 한번 말씀해 주세요"

### 3. 발음 집중 훈련
**우선순위: 중간**

#### 3.1 한국어 특수음 훈련
- **종성 연습**: 받침 발음 (ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅇ)
- **이중모음**: 야, 여, 요, 유, 와, 워, 위, 의
- **경음/격음**: ㅂ-ㅍ-ㅃ, ㄱ-ㅋ-ㄲ, ㄷ-ㅌ-ㄸ

#### 3.2 억양 패턴 학습
- 의문문 억양 ↗
- 평서문 억양 ↘  
- 감탄문 억양

### 4. 게이미피케이션 요소
**우선순위: 중간**

#### 4.1 성취 시스템

#### 4.2 진도 추적

- **발음 점수 그래프**: 시간별 향상도
- **카테고리별 진도**: 직업/상황별 완료율

### 5. 개인화 학습 시스템
**우선순위: 낮음**

#### 5.1 난이도 자동 조절
- 사용자 발음 점수 분석
- 약점 영역 집중 학습
- 맞춤형 복습 콘텐츠

#### 5.2 학습 기록 관리
- **음성 녹음 저장**: 발음 변화 추적
- **오답 노트**: 자주 틀리는 표현 모음
- **개인 단어장**: 어려운 단어 북마크

### 6. 멀티미디어 학습 지원
**우선순위: 낮음**

#### 6.1 시각적 학습 자료
- **입 모양 애니메이션**: 정확한 발음 위치 표시
- **상황별 이미지**: 시각적 맥락 제공
- **한글 자모 연습**: 글자별 발음 가이드

#### 6.2 문화 컨텍스트 교육
- **존댓말 사용법**: 상황별 높임 표현
- **한국 직장 문화**: 기본 예의와 매너
- **지역별 방언**: 주요 지역 사투리 소개

## 📊 성공 지표 (KPI)

### 학습 효과성
- **발음 점수 향상률**: 평균 20점 이상 향상
- **학습 완료율**: 시작한 코스의 70% 이상 완료
- **정확도 달성률**: 80점 이상 달성 비율

### 사용자 참여도
- **일일 활성 사용자**: DAU 목표치
- **평균 학습 시간**: 세션당 15분 이상
- **재방문율**: 주간 3회 이상

### 비즈니스 임팩트
- **사용자 만족도**: 4.5/5.0 이상
- **추천율**: NPS 50 이상
- **학습 성과**: 실제 업무/생활 적용 사례

## 🔧 기술적 고려사항

### 데이터베이스 설계
```sql
-- 학습 콘텐츠 테이블
CREATE TABLE learning_content (
  id INT PRIMARY KEY,
  category VARCHAR(50),      -- 직업/상황 분류
  difficulty INT,            -- 난이도 (1-5)
  text TEXT,                -- 목표 문장
  phonetic TEXT,            -- 발음 표기
  translation TEXT,         -- 번역
  audio_url VARCHAR(255),   -- TTS 오디오 파일
  created_at TIMESTAMP
);

-- 사용자 학습 기록
CREATE TABLE user_progress (
  id INT PRIMARY KEY,
  user_id INT,
  content_id INT,
  score INT,                -- 발음 점수
  attempts INT,             -- 시도 횟수
  audio_file VARCHAR(255),  -- 사용자 음성 파일
  feedback TEXT,            -- AI 피드백
  completed_at TIMESTAMP
);
```

### API 확장
```javascript
// 학습 콘텐츠 API
GET /api/content?category=construction&difficulty=1
POST /api/progress           // 학습 기록 저장
GET /api/user/stats         // 개인 통계
GET /api/achievements       // 성취 현황

// 새로운 평가 API 파라미터
POST /api/evaluate {
  target: "목표 문장",
  asrText: "인식 결과", 
  category: "construction",    // 추가: 분야별 평가
  userId: "user123",          // 추가: 개인화
  difficulty: 2               // 추가: 난이도
}
```

### 확장성 고려
- **콘텐츠 CMS**: 관리자가 쉽게 학습 자료 추가
- **다국어 지원**: 베트남어, 중국어 등 모국어 번역
- **오프라인 모드**: PWA로 오프라인 학습 지원
- **음성 합성**: 목표 문장 TTS 재생
