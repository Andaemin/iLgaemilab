# 🔧 5주 MVP 기능 범위 (우선순위 중심)

### **✅ 1. STT-TTS 기반 음성 대화 시스템**

-   마이크 입력 → 음성 인식(STT) → 대화 생성(GPT) → 음성 출력(TTS)

### **✅ 2. GPT 기반 역할극 대화 시나리오**

-   실시간 시나리오 기반 대화 흐름 구현
-   오답 시 즉시 피드백 제공

### **✅ 3. 간단한 3~5분 레벨테스트**

-   듣기·읽기·말하기 중심 평가
-   자동 채점 후 학습 경로 추천

### **✅ 4. 직무 Quick Start 카드**

-   “오늘 꼭 써야 할 5문장”을 카드 형태로 제공
-   음성 반복 학습 기능 포함

### **✅ 5. 간단한 개인화 추천 기능**

-   직무 태그 + 레벨 기반 콘텐츠 추천
-   사용자의 학습 로그 기반 콘텐츠 큐레이션
